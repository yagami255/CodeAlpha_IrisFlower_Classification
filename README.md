# Iris Flower Classification

This repository contains a machine learning project that classifies Iris flowers into three species: **Iris Setosa**, **Iris Versicolor**, and **Iris Virginica**. The model uses flower measurements such as sepal length, sepal width, petal length, and petal width to predict the correct species.

## Project Description

The Iris dataset is one of the most popular datasets used for learning machine learning classification. This project demonstrates a complete supervised machine learning workflow, starting from data loading and exploration to model training, evaluation, and prediction.

The project includes exploratory data analysis, data visualization, preprocessing, and comparison of multiple classification algorithms. It is beginner-friendly and useful for understanding how machine learning models are built and evaluated using Python.

## Features

- Load and explore the Iris dataset
- Analyze flower measurements and species distribution
- Visualize data using graphs, pair plots, and heatmaps
- Preprocess data using encoding and scaling
- Split data into training and testing sets
- Train multiple machine learning models
- Compare model performance
- Evaluate results using accuracy, confusion matrix, and classification report
- Predict Iris species from new input data

## Machine Learning Models Used

- Logistic Regression
- K-Nearest Neighbors
- Decision Tree
- Random Forest
- Support Vector Machine

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset

The project uses the Iris dataset, which contains 150 flower samples from three Iris species. Each sample includes four numerical features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/iris-flower-classification.git
```

Navigate to the project folder:

```bash
cd iris-flower-classification
```

Install the required libraries:

```bash
pip install pandas numpy scikit-learn matplotlib seaborn
```

## How to Run

Open the Jupyter Notebook:

```bash
jupyter notebook iris_classification.ipynb
```

Run all cells step by step to see the complete workflow, including data analysis, model training, evaluation, and prediction.

## Results

The trained models are evaluated using accuracy scores and classification metrics. The project shows that petal length and petal width are highly important features for identifying Iris flower species.

## Conclusion

This project is a simple and practical example of a machine learning classification problem. It is suitable for beginners who want to learn data analysis, visualization, preprocessing, model training, and evaluation using Python.
